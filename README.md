# Task Management Application

A simple and responsive Task Management Web Application built using HTML, CSS, and JavaScript. This application helps users organize their daily tasks by allowing them to add, edit, complete, and delete tasks with an intuitive user interface.

## Features

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Responsive design for desktop and mobile devices
* User-friendly interface
* Real-time task updates

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Pranav5674/SCT_WD_4.git
```

### Run the Application

1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Start managing your tasks.

## How to Use

1. Enter a task in the input field.
2. Click the Add button to create a new task.
3. Edit tasks whenever needed.
4. Mark tasks as completed once finished.
5. Delete tasks that are no longer required.
